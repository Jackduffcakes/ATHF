About Applesauce

Applesauce is easily digestable and safe to eat when having indigestion issues. Applesauce has soluble fiber which studies have shown to help reduce diarrhea. It doesn't have any insoluble fiber which can make diarrhea or indigestion worse and it also has pectin which has been shown to improve diarrhea symptoms.
Sources

(Homann et al., 1994), (Nakao et al., 2002), (Rabbani et al., 2001), (Triplehorn and Millard, 2002)
